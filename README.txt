This is server-side rendering application written in PHP. JavaScript is mostly implemented here for AJAX and live updates for clients, and also it uses MVC architecture pattern.It represents app for reviewing movies, where you can write comment and rate a movie that you want. It also has an admin panel section, where admin can change anything stored in database.
Live version is available at:
https://movielense.000webhostapp.com/

And there is also documentation written in Serbian.
